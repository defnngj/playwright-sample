# playwright-sample

## 什么是Playwright?

官网：https://playwright.dev/

Playwright可以在所有浏览器中实现快速、可靠和强大的自动化测试。

## playwright 安装

1. 安装 [node.js](https://nodejs.org/zh-cn/)

```shell
$ node -v
v12.18.2
```

2. 安装[playwright](https://github.com/microsoft/playwright)

```shell
$ npm i -D playwright
```

## 运行sample

```shell
$ node .\sample\first_script.js
```
